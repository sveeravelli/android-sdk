public interface AdsManager

An object which handles playing ads after they've been received from the server. After the AdsManager is retrieved from an AdsManagerLoadedEvent, init should be called first. After the LOADED event is fired, start can be called. At any time, destroy can be called to end ad playback.

Public Methods

public abstract void addAdErrorListener (AdErrorEvent.AdErrorListener errorListener)

Registers a listener for errors that occur during the ad initialization and playback.

public abstract void addAdEventListener (AdEvent.AdEventListener adEventListener)

Registers a listener for ad events that occur during ad initialization and playback.

public abstract void destroy ()

Stops the ad and all tracking, then releases all assets that were loaded to play the ad.

public abstract List<Float> getAdCuePoints ()

List of content time offsets in seconds at which ad breaks are scheduled. The list will be empty if no ad breaks are scheduled.

public abstract Ad getCurrentAd ()

Get currently playing ad.

public abstract void init ()

Initializes the ad experience on the ads manager using default rendering settings.

public abstract void init (AdsRenderingSettings settings)

Initializes the ad experience on the ads manager. This will load an ad and eventually send the LOADED event, indicating the ad is ready to be started. If an ads list is loaded, this method will start playback.

public abstract void removeAdErrorListener (AdErrorEvent.AdErrorListener errorListener)

Removes a listener for error events.

public abstract void removeAdEventListener (AdEvent.AdEventListener adEventListener)

Removes a listener for ad events.

public abstract void skip ()

Skip the current ad.

public abstract void start ()

Starts playing the ads. This will be ignored for ads lists, because init() will start playback.
