Contains the VideoAdPlayer interface which should be extended to create an SDK compatible video player. Documentation on creating this can be found here.

Interfaces

VideoAdPlayer Defines the set of methods that a video player must implement to be used by the SDK, as well as a set of callbacks that it must fire. 
VideoAdPlayer.VideoAdPlayerCallback Callbacks that the player must fire. 

Classes

VideoProgressUpdate Defines an update to the video's progress.
