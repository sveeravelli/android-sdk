Public Methods

public abstract AdDisplayContainer getAdDisplayContainer ()

Returns the container that will be used to display ads.

public abstract String getAdTagUrl ()

Returns the URL from which ads will be requested.

public abstract String getAdsResponse ()

Returns the canned ads response.

public abstract String getExtraParameter (String key)

Returns an extra parameter set by the setExtraParamaters method.

public abstract Map<String, String> getExtraParameters ()

Returns a map of all extra parameters that have been set.

public abstract Object getUserRequestContext ()

Returns the user-provided object that is associated with the ads request.

public abstract void setAdDisplayContainer (AdDisplayContainer adDisplayContainer)

Sets the container that will be used to display ads. Within the AdDisplayContainer, non-linear ads will be placed in the container, linear ads will be played in the player, and companion ads will be placed in the companion slots.

public abstract void setAdTagUrl (String adTagUrl)

Sets the URL from which ads will be requested. This parameter is required.

public abstract void setAdsResponse (String cannedAdResponse)

Set the canned ads response.

public abstract void setExtraParameter (String key, String value)

Sets an extra parameter to be used by the SDK. This is only used for extra features within the SDK and is reserved for future use. This parameter is optional.

public abstract void setUserRequestContext (Object userRequestContext)

Sets the user-provided object that is associated with the ads request. If supplied here, it will be available once the ads load is complete or fails. This object can be used to identify the request that caused an error or which request loaded a particular ads manager.
